# CarWash
# GetWash — Платформаи ҷустуҷӯ ва фармоиши автомойкаҳо

Ин веб-сайт як платформаи муосир ва адаптивӣ барои замимаи мобилии **GetWash** мебошад, ки раванди пайдо кардани автомойкаи холӣ ва банд кардани вақтро осон ва автоматӣ мекунад. Лоиҳа бо риояи тамоми стандартҳои муосири веб-дизайн ва адаптивӣ (Responsive) сохта шудааст.

## 📱 Хусусиятҳои асосии лоиҳа (Структура)

Лоиҳа аз қисмҳои зерини функсионалӣ иборат аст, ки барои ҳамаи экранҳо (Мобилӣ, Планшет ва Компютер) комилан мувофиқ карда шудаанд:

### 1. Қисми Асосӣ (Hero Section)
* Ҷустуҷӯи зуд ва брони замима дар вақти дилхоҳ.
* Тугмаи фаъоли "Получить доступ" барои гузариш ба марҳилаи навбатӣ.

### 2. Оғози Зуд (Быстрый старт)
Шарҳи зина ба зинаи истифодаи барнома:
* Боргирӣ ва бақайдгирӣ (App Store / Google Play).
* Интихоби наздиктарин автомойка дар харита.
* Интихоби хизматрасониҳои иловагӣ (шустани дохил, сайқалдиҳӣ ва ғ.).
* Пардохти онлайн ва беламс дар дохили замима.

### 3. Бахши Шарикон (Стань нашим партнером)
Имкониятҳо барои соҳибони автомойкаҳо барои автоматӣ кардани тиҷорат ва ҷалби мизоҷони нав:
* Назорати таърихи фармоишҳо.
* Сохтани ҳисоботҳои ҳафтаина/моҳона/солона.
* Алоқаи мустақим бо мизоҷон.
* Назорати сифати кори кормандон.

### 4. Баннери Боргирӣ (Call to Action)
* Даъват ба истифодаи барнома барои сарфаи вақт ва пешгирӣ аз навбатпоиҳо.
* Тугмаҳои махсуси сифати баланд барои гузариш ба **App Store** ва **Google Play**.

### 5. Шарҳҳо ва Футер (Отзывы и Footer)
* Шарҳҳои воқеии корбарон ва ронандагон дар шакли слайдери адаптивӣ.
* Футери тоза ва тира бо пайвандҳои муҳим ва логотипи расмии бренд.
